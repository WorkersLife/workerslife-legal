Datenschutzerklärung

Gültig für: WorkersLife Welcome Discord Bot

Verantwortlich:
Workerslife

1. Allgemeines

Der WorkersLife Welcome Bot ist ein Discord-Bot, der automatisch eine Willkommensnachricht sendet, wenn ein Nutzer dem offiziellen Workerslife Discord-Server beitritt.

2. Erhobene Daten

Der Bot verarbeitet ausschließlich folgende Informationen:
- Benutzername
- Profilbild (Avatar-URL)

Diese Informationen stammen direkt von Discord und werden nicht dauerhaft gespeichert. Sie dienen ausschließlich dazu, eine personalisierte Willkommensnachricht zu generieren.

3. Keine Weitergabe an Dritte

Es erfolgt keine Weitergabe von Daten an Dritte. Es werden keine externen Dienste oder Drittanbieter verwendet.

4. Speicherort und -dauer

Der Bot speichert keine personenbezogenen Daten dauerhaft. Alle verwendeten Informationen werden nur zur Laufzeit verarbeitet und danach verworfen.

5. Rechte der Nutzer

Da keine personenbezogenen Daten dauerhaft gespeichert werden, besteht kein Anspruch auf Auskunft, Berichtigung oder Löschung. Für alle datenschutzrechtlichen Anliegen kann sich der Nutzer jedoch an Discord oder Workerslife wenden.
